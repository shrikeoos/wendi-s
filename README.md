# wendi-s

A small pixel art browser game built as a birthday surprise.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local development

```sh
# Clone the repo
git clone https://github.com/shrikeoos/wendi-s.git
cd wendi-s

# Install dependencies
npm install

# Start the dev server
npm run dev
```

## Deploy

```sh
npm run deploy
```

Publishes to GitHub Pages at `https://shrikeoos.github.io/wendi-s/`.
